# Dashboard App (Next.js)

Use this folder for the neon command center (Next.js 14 + App Router + Tailwind).

Suggested commands:
```
npx create-next-app@latest dashboard --ts --app --src-dir --tailwind --eslint --use-npm
```

Then move generated files into this folder. Wire it to Supabase and reuse `components/neon-dashboard-ui.tsx`.
